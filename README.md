# duckducko-cli
DuckDuckGo search engine for CLI

## One-line installer
```bash
$ sudo wget https://raw.githubusercontent.com/OdinF13/duckduckgo-cli/main/duckduckgo.sh -O /usr/bin/duckduckgo && sudo chmod +x $_
```
